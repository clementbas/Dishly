# Dishly 🍽️

Application web fullstack de partage et découverte de recettes de cuisine.

## Stack

- **Frontend** : React + Vite + Tailwind CSS
- **Backend** : Node.js + Express
- **Base de données** : MongoDB + Mongoose

## Structure

```
Dishly/
├── frontend/   # React app
├── backend/    # Express API
└── README.md
```

## Lancement

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
